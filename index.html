Data Overview

This visualization presents employment statistics for all Australian states and territories, including employment rates, labour force size, and population data from the latest Australian Bureau of Statistics release for Q3 2025.

Visual Encoding

Circle size represents labour force size, while color intensity indicates employment rates. Hover over each circle to see detailed statistics for each jurisdiction.

Key Insights

The map reveals significant variations in employment patterns across Australia, with larger states showing different employment dynamics compared to smaller territories and remote regions.

Technical Details

Built with Vega-Lite using a Conic Equal Area projection optimized for Australia. Interactive tooltips provide comprehensive employment metrics for each region.
